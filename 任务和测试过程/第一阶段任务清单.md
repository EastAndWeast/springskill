# 春节续命网站 - 任务清单与测试记录

## 核心技术栈选择
为了遵守 KISS 原则，我们采用开箱即用、轻量且强大的极简栈：
- **前端框架**：React + Vite (组件化方便处理模块，速度极快)
- **UI & 样式**：Tailwind CSS v4 (避免写冗长 CSS，直接用 Utility Classes)
- **图标**：Lucide-react (现代化、极简线条风格)
- **状态持久化**：`localStorage` (纯前端实现，无须后端，保证用户隐私极简开发)
